# 🚗 ParkVik – Smart Parking System

## 📌 Overview

ParkVik is a smart parking management system developed using Python and Tkinter. The application provides a simple, efficient, and user-friendly graphical interface that allows users to manage parking slots digitally. It is designed as a simulation of a real-world smart parking solution, helping reduce time, effort, and congestion caused by manual parking systems.

With increasing urbanization and vehicle usage, finding parking spaces has become a major challenge. ParkVik addresses this issue by offering a digital platform where users can quickly view available parking slots, book them, and manage their bookings with ease.

---

## 🎯 Objectives

- To design a simple and interactive parking system  
- To reduce time spent searching for parking slots  
- To demonstrate real-world problem solving using Python  
- To provide a user-friendly GUI-based application  

---

## ✨ Key Features

### 🔐 User Authentication
- New users can register with a username and password  
- Existing users can log in securely  
- User data is stored using JSON files  

### 🅿️ Parking Slot Management
- Displays 10 parking slots in a grid format  
- Slots are color-coded:
  - 🟢 Green → Available  
  - 🔴 Red → Already booked  
  - 🔵 Blue → Booked by current user  

### ✅ Slot Booking
- Users can book a slot by simply clicking on it  
- Includes a **payment simulation (₹50)**  
- Updates slot status instantly  

### ❌ Cancel Booking
- Users can view their booked slots  
- Cancel any booking with one click  
- Slot becomes available again  

### 🎨 User Interface
- Built using Tkinter GUI  
- Mobile-style layout (360x640 window)  
- Attractive colors and design  
- Hover effects on buttons  
- Smooth fade-in animation  
- Image/logo support
